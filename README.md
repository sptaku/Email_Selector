# 📧 Email Selector

メールアドレスリストのフィルタリングツール

## 概要

2つのメールアドレスリストを比較し、一方のリストから他方に含まれるアドレスを除外するWebアプリケーションです。

## 機能

- **リストA（メインリスト）**: フィルタリング対象のメールアドレスを入力
- **リストB（除外リスト）**: 削除したいメールアドレスを入力
- **リストC（出力結果）**: Aに含まれ、Bに含まれないメールアドレスを出力

### 主な特徴

- 🔒 **プライバシー保護**: すべての処理はブラウザ内で完結し、サーバーへのデータ送信は一切ありません
- 📝 **名前の自動除去**: `山田太郎 <taro@example.com>` のような形式からメールアドレスのみを抽出
- 🔄 **重複の自動除去**: リストAに同じアドレスが複数含まれていても、出力結果はユニーク（重複なし）になります
- 🔤 **並べ替え機能**: 出力結果をアルファベット順（A→Z）、逆順（Z→A）、元の順序で並べ替え可能
- ⬇️ **ダウンロード機能**: 結果をテキストファイルとしてダウンロード可能
- 📋 **コピー機能**: ワンクリックでクリップボードにコピー
- 📊 **統計表示**: 入力件数、除外件数、出力件数をリアルタイム表示

## 使い方

1. **リストA**にフィルタリング対象のメールアドレスを1行ずつ入力
2. **リストB**に除外したいメールアドレスを1行ずつ入力
3. 「**フィルタリング実行**」ボタンをクリック
4. 結果は**リストC**に表示されます
5. 必要に応じて「**ダウンロード**」または「**コピー**」ボタンを使用

## 対応形式

以下の形式のメールアドレスに対応しています：

```
taro@example.com
山田太郎 <hanako@example.com>
"佐藤花子" <sato@test.org>
info@company.co.jp
```

すべての形式からメールアドレス部分のみを抽出して処理します。

## プライバシーについて

⚠️ **重要**: このツールは完全にクライアントサイドで動作します。

- 入力されたメールアドレスは**一切サーバーに送信されません**
- すべての処理は**ブラウザ内で完結**します
- ページを閉じると、入力されたデータは**すべて破棄**されます
- Cookie、LocalStorage、SessionStorageへの保存は**行いません**

## 技術仕様

- **HTML5 / CSS3 / JavaScript（Vanilla JS）**
- 外部ライブラリ不要（フォント読み込みを除く）
- レスポンシブデザイン対応
- モダンブラウザ対応（Chrome, Firefox, Safari, Edge）
